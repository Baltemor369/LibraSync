# **LibraSync**

A easy use Manager for your books collection.

## **Requierements**

- **Python 3.11.1**

## **Features**

- add a book

- delete a book

- modify a book

- sort list of books

- search a book by its name/author/Family

- settings menu

- save your data in a file
